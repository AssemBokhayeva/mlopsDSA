## Проект в рамках обучения в школе DSA
Bank Marketing ML App

### Описание проекта

Это веб-приложение использует методы машинного обучения для прогнозирования того, подпишется ли клиент банка на депозит после маркетингового контакта.

Датасет: bank.csv (открытые источники). Он содержит информацию о клиентах, их финансовых характеристиках и результатах прошлых маркетинговых кампаний.

Приложение реализует полный MLOps pipeline: подготовка данных (в предудщих уроках было реализовано) → обучение модели → REST API → веб-интерфейс → контейнеризация через Docker.

### Задача проекта

Тип задачи: бинарная классификация

Целевая переменная: y — подписался ли клиент на депозит (yes / no)

Признаки, используемые моделью:

1) age -	Возраст клиента
2) balance	Баланс на счёте
3) duration	Длительность последнего контакта
4) contacts	Количество контактов в кампании

### Практическая польза:

Сокращение неэффективных звонков

Повышение конверсии маркетинговых кампаний

Оптимизация работы колл-центра

### Интерфейс приложения

Приложение реализовано через Streamlit:

Форма ввода данных клиента включает: Возраст, Баланс на счёте, Длительность последнего звонка, Количество контактов в текущей кампании
Кнопка: Predict
Результат после нажатия:
Client will subscribe / Client will not subscribe

### Запуск проекта локально

#### Установить зависимости:
pip install -r req.txt

#### Обучить модель:

python model/train_model.py

#### Запустить API:

uvicorn api.app:app --reload

#### В новом терминале запустить интерфейс:

streamlit run app/streamlit_app.py
#### Запуск через Docker

Собрать Docker image:

docker build -t bank-ml-app .

Запустить контейнер:

docker run -p 8000:8000 bank-ml-app
